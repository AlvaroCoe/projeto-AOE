Sistema de Escala:

Criado um programa em react com java e javascript para gerenciar a escala de funcionários em plataformas. O sistema permite cadastrar, listar, editar as informações e excluir os registros, tudo direto em uma única tela para facilitar o controle.

🛠️ O que o programa faz:

Cadastrar: Você preenche os dados do funcionário no formulário e ele é salvo no sistema;

Listar: Mostra todos os profissionais organizados por plataforma automaticamente;

Editar: Ao clicar em editar, os dados do funcionário voltam para o formulário para você alterar e salvar;

Deletar: Um botão simples que pede confirmação e apaga o registro do sistema;
